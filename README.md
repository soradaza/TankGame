・操作方法<br>
wキー：前進<br>
sキー：後退<br>
aキー：左に旋回<br>
dキー：右に旋回<br>
スペースキー：攻撃<br>
左クリック：上昇<br>
右クリック：降下<br>
マウス：攻撃照準<br>
